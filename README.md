# 📊 Graphs App

This is a lightweight Android app built with Kotlin and Jetpack Compose to display different types of charts:

- 🟠 Pie Chart  
- 🟦 Bar Chart  
- 🕯 Candle Stick Chart  

---

## 🔧 Built With

- Kotlin
- Jetpack Compose
- MPAndroidChart
